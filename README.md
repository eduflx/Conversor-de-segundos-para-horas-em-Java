# Conversor-de-segundos-para-horas-em-Java
Programa em Java que converte segundos para horas, minutos e segundos.
O Programa é baseado em 3 funções dentro da classe:

    Calculo em horas: Divide os segundos para a quantidade respectiva em horas.
    Calculo em minutos: Pega o restante da divisão e divide para a quantidade respectiva em minutos.
    Calculo em segundos: Pega o restante.

O programa retorna os resultados dentro da classe principal, que solicita o usuário a entrada e executa as funções.

Exercicio 4 da Lista de exercícios nº3 de Extrutura de Dados 1 do curso de ADS do IFTM Campus Patrocínio.
